# HackLinux Academy

Interactive Linux command trainer with a terminal-style interface.

## Demo

https://hacklinux-academy.vercel.app

## Features

- Linux theory lessons
- Terminal practice simulator
- Beginner, Intermediate and Advanced levels
- Progress saving in the browser
- Locked levels
- Profile and result pages

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Vercel

## Run Locally

```bash
git clone https://github.com/332mindset/hacklinux-academy.git
cd hacklinux-academy
npm install
npm run dev
